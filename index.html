Technical Skills

Projects

NASA APOD

Using the NASA Astronomy Photo of the Day API, ten random images with their descriptions are displayed on load with the ability to load more on button press. The user is then able to add and remove photos from their favorites list respectedly. Technologies involved within the project are vanilla JS, CSS and HTML.

Loop Studios Landing Page

From Frontend Mentor, this landing page is developed primarily with HTML, CSS and a bit of vanilla JS to toggle menu elements. It is mobile responsive across various platforms.

Sinnoh Card List

Developed using the Pokemon API, Pokemon cards from the Sinnoh Region are dynamically generated through vanilla JS and displayed with CSS Grid. From the API, a the names, measurements, moves, abilties, and images of those Pokemon are retrieved. The site is mobile responsive as well.

Form Validator

A mobile responsive site in which the user enters their information within the form and the data is verified with the Javascript Validation API. The site was built with vanilla JS, HTML and CSS.
